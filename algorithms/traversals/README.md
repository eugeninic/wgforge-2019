### Условие 

Реализовать 6 функций на любом языке программирования, которые делают три обхода дерева в рекурсивной и нерекурсивной реализации

### Traversals:

* inorder
* postorder
* level order traversal (bfs)

### Class __Node__ looks like:

```
Node {
    T key;
    Node Parent;
    Node FirstChild;
    Node NextSibling;
}
```
